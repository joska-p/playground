# Playground (Astro)

The main Astro site hosting all experiments.

---

## Commands

From workspace root:

| Command | Action |
|---------|--------|
| `pnpm dev` | Start dev server at `localhost:4321` |
| `pnpm build` | Build to `./dist` |
| `pnpm preview` | Preview build locally |
