# @repo/playground

> Astro site for the Creative Playground — documentation, experiments, and the notebook.

Built with **Astro 6**, **Tailwind CSS v4**, and **React 19**.

## Develop

```bash
pnpm --filter @repo/playground dev
```

## Build

```bash
pnpm --filter @repo/playground build
```
