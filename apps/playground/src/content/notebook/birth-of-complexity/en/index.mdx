</h2>

<br />

<TheSkeptic>We must begin somewhere.</TheSkeptic>

<TheNarrator>Mathematics starts from a simple idea:</TheNarrator>

<TheDreamer>What if something exists?</TheDreamer>

<br />

<StepOne />

<br />

<TheNarrator>Then we add a single rule.</TheNarrator>

<TheNarrator>If a number exists, its successor also exists.</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$1 + 1 =$ something</span>
</div>

<br />

<TheSkeptic>The mechanism is as simple as it is tireless.</TheSkeptic>

<TheNarrator>From one brick and one generative rule, an infinite set unfolds.</TheNarrator>

<figure title="the Natural Numbers" class="flex items-center justify-start gap-4 pl-4">
  <figcaption class="text-accent/80">the Natural Numbers</figcaption>
  <span class="text-accent text-5xl">$$\mathbb{N}$$</span>
</figure>

<br />

<TheNarrator>Then, multiplication is not a new invention.</TheNarrator>

<TheSkeptic>It is just a compact notation for repeated addition.</TheSkeptic>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4"> $$1 + 1 + 1 = 1 * 3$$</span>
</div>

<br />

<br />

<br />

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Step 01</span>
  <span class="shrink-0">Zero and the Negatives</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$\mathbb{Z}$$</span>
</h2>

<br />

<TheDreamer>What if we undo the addition?</TheDreamer>

<TheSkeptic>If we can add, we must be able to reverse the operation.</TheSkeptic>

{/* prettier-ignore-start */}

<TheNarrator>
  We invent subtraction. And for a while, everything holds: all results land safely in $$\mathbb{N}$$.
</TheNarrator>

{/* prettier-ignore-end */}

<TheNarrator>Then someone dares to write:</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$1 - 1 = something$$</span>
</div>

<TheNarrator>Silence.</TheNarrator>

<TheDreamer>The result should exist.</TheDreamer>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$1 - 2$$ is worse still</span>
</div>

<TheSkeptic>The answer lives nowhere in our universe.</TheSkeptic>

<br />

<TheNarrator>
  We have only one option: enlarge it. We invent a new symbol for the void $0$, and a new direction
  for the negatives $-1$.
</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$0 = n - n$$ and $$0 - n = -n$$</span>
</div>

<TheNarrator>And we give the **negative numbers** somewhere to live.</TheNarrator>

<figure title="the Natural Numbers" class="flex items-center justify-start gap-4 pl-4">
  <figcaption class="text-accent/80">the integers</figcaption>
  <span class="text-accent text-5xl">$$\mathbb{Z}$$</span>
</figure>

<br />

<br />

<br />

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Step 01</span>
  <span class="shrink-0">The Rational Numbers</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$\mathbb{Q}$$</span>
</h2>

<br />

<TheDreamer>What if we undo a multiplication?</TheDreamer>

<TheNarrator>
  The same reasoning repeats. If we can multiply, we must be able to invert the operation.
</TheNarrator>

<TheNarrator>
  We invent division. And once again at the beginning, everything works perfectly.
</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$4 \div 2 = 2$$</span>
</div>

<TheDreamer>But what if it doesn't divide cleanly?</TheDreamer>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$n = 1 \div 2$$</span>
</div>

<TheSkeptic>
  The space between the integers is empty. There are no positions between one and two.
</TheSkeptic>

<TheNarrator>We invent new symbols to describe the pieces.</TheNarrator>

{/* prettier-ignore-start */}

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">
    $$\dfrac{a}{b}$$, where $$a, b \in \mathbb{Z}$$ and $$b \neq 0$$.
  </span>
</div>

{/* prettier-ignore-end */}

<TheNarrator>These new numbers need a home.</TheNarrator>

<TheDreamer>The number line becomes dense.</TheDreamer>

<figure title="the Rational Numbers" class="flex items-center justify-start gap-4 pl-4">
  <figcaption class="text-accent/80">the rational numbers</figcaption>
  <span class="text-accent text-5xl">$$\mathbb{Q}$$</span>
</figure>

<br />

<br />

<br />

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Step 04</span>
  <span class="shrink-0">The Real Numbers</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$\mathbb{R}$$</span>
</h2>

<br />

<TheNarrator>From multiplication, we derive exponentiation.</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$3 ^ 9 = 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3$$</span>
</div>

<TheSkeptic>It is just a fancy notation.</TheSkeptic>

<br />

<TheDreamer>What if we undo the exponentiation?</TheDreamer>

<TheNarrator>We build a tool for it: the symbol $$\sqrt{}$$.</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$\sqrt{a} = b$$ means that $$b^2 = a$$</span>
</div>

<br />

<TheNarrator>For a while it was fine. Then a crack appeared in the geometry.</TheNarrator>

<TheDreamer>What if we undo the square of 2?</TheDreamer>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$n = \sqrt{2}$$</span>
</div>

<TheSkeptic>
  We checked every fraction. No two integers can be divided to produce this ratio. It cannot be
  written.
</TheSkeptic>

<TheNarrator>The Real Numbers are born to fill the infinite gaps left behind.</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$\sqrt{2}, \pi, e, 3.33333333...$$ are real numbers</span>
</div>

<figure title="the Real Numbers" class="flex items-center justify-start gap-4 pl-4">
  <figcaption class="text-accent/80">the real numbers</figcaption>
  <span class="text-accent text-5xl">$$\mathbb{R}$$</span>
</figure>

<br />

<br />

<br />

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Step 05</span>
  <span class="shrink-0">The Imaginary Unit</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$\mathbb{i}$$</span>
</h2>

<br />

<TheNarrator>
  We have noticed that when you square a real number, the result is always positive. Even if you
  square a negative number, the result is still positive.
</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$(-2)^2 = 4$$ or, $$(-x)^2 = x^2$$</span>
</div>

<TheNarrator>
  It enforces one absolute law of our reality: <br />
  Every real square is $\geq 0$.
</TheNarrator>

<TheDreamer>What if we allow a square to be a negative number?</TheDreamer>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$x^2 = -1$$</span>
</div>

<TheSkeptic>The equation x^2 + 1 = 0 has no real solution.</TheSkeptic>

<TheNarrator>
  Mathematicians chose to act rather than retreat. They postulated the existence of a number $$i$$
  such that:
</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$i^2 = -1$$</span>
</div>

<TheNarrator>
  Not a trick. Not a shorthand for something else. A legitimate new entity born from a necessity.
</TheNarrator>

<br />

<figure title="the imaginary numbers" class="flex items-center justify-start gap-4 pl-4">
  <figcaption class="text-accent/80">the imaginary numbers</figcaption>
  <span class="text-accent text-5xl">$$\mathbb{i}$$</span>
</figure>

<br />

<br />

<br />

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Step 06</span>
  <span class="shrink-0">The Complex Plane</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$\mathbb{C}$$</span>
</h2>

<br />

<TheDreamer>Where is $i$?</TheDreamer>

<TheNarrator>
  $i$ is not in the real world. It does not lie anywhere on our line. We need to be creative. We
  create a new dimension.
</TheNarrator>

<TheSkeptic>We visualize it as a line perpendicular to the real axis.</TheSkeptic>

<TheNarrator>
  And now with those two axes, the real number line extends into a plane. This is where the complex
  numbers live: $$\mathbb{C}$$
</TheNarrator>

<ComplexPlane />

<TheNarrator>Every complex number takes the form:</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$Z = a + ib \qquad a,\, b \in \mathbb{R}$$</span>
</div>

<TheNarrator>
  where $a$ is the **real part** and $b$ is the **imaginary part**. The line has become a plane. A
  brand new world.
</TheNarrator>

<br />

<figure title="the complex numbers" class="text-center">
  <span class="text-accent text-5xl">$$\mathbb{C}$$</span>
  <figcaption class="text-accent/80">the complex numbers</figcaption>
</figure>

<h2 class="text-primary my-8 flex items-center gap-4 text-2xl font-bold tracking-tight sm:text-3xl">
  <span class="text-muted-foreground/50 shrink-0 self-start text-sm font-medium">Epilogue</span>
  <span class="shrink-0">Euler's Formula</span>
  <div class="bg-border h-px flex-grow"></div>
  <span class="text-accent shrink-0">$$e^{i\pi} = -1$$</span>
</h2>

<br />

<TheDreamer>What if we bring them all together?</TheDreamer>

<TheNarrator>Five symbols. Five different origins. One equation.</TheNarrator>

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">$$0 \qquad 1 \qquad \pi \qquad e \qquad i$$</span>
</div>

<br />

- **$0$** — the void, born when subtraction outran the naturals.
- **$1$** — the origin, the first postulate, the brick.
- **$\pi$** — the ratio of a circle's circumference to its diameter, irrational and inevitable.
- **$e$** — the base of natural growth, the number that is its own derivative.
- **$i$** — the answer to the impossible square.

<TheNarrator>
  They come from different centuries, different problems, and different branches of mathematics.
  They have no obvious reason to meet.
</TheNarrator>

<TheSkeptic>And yet.</TheSkeptic>

<figure title="Euler's Formula" class="text-center">
  <span class="text-accent text-5xl">$$e^{i\pi} + 1 = 0$$</span>
  <figcaption class="text-accent/80">Euler's Identity</figcaption>
</figure>
