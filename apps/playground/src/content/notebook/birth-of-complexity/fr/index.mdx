## L'Émergence de la Complexité

Ceci est une invitation à la promenade. À chaque carrefour, nous poserons la même question : **et si on inversait cela ?** C'est cette question, posée encore et encore, qui construit l'architecture entière des nombres à partir du sol.

## Les Nombres Naturels

**La Question**

- Et si quelque chose existait ?

Il faut bien commencer quelque part. Les mathématiques commencent par une idée d'une simplicité désarmante : que _quelque chose existe_. Ce quelque chose est $1$.

Il faut ajouter une règle de mouvement :
- **Si un nombre existe, son successeur existe aussi.**

```latex
Appliquez-la une fois     1 + 1 = 2
Encore                    2 + 1 = 3
Puis                      4, 5, 6, 7, 8, 9, vers l'infini
```

- Le mécanisme est aussi simple qu'infatigable.

**La Conséquence**

- D'une brique et d'une règle générative, un ensemble infini émerge.
- La **multiplication** n'a jamais été une nouvelle règle.
- C'est une notation compacte pour l'addition répétée, déjà contenue dans le successeur depuis le commencement.

## Zéro et les Nombres Négatifs

**La Question**

- Peut-on annuler une addition ?

Si nous pouvons ajouter, nous demandons : _peut-on inverser l'opération ?_

Avancer de 3 pas et revenir au point de départ

- Quelle opération cela décrit-il ?

Nous inventons la soustraction.

```latex
Pendant un temps, tout va bien : 5 - 2 = 3, 7 - 4 = 3
```

Tous les résultats atterrissent dans $\mathbb{N}$.

```latex
Puis quelqu'un ose écrire 3 - 3. Silence.
```

Le résultat _devrait_ exister.
On avance 3, on recule 3, on est de retour à l'origine.
Mais $\mathbb{N}$ n'a **aucun symbole pour le rien**.

```latex
Et 3 - 5 est pire: la réponse ne vit nulle part dans notre univers.
```

L'univers n'est **pas clos** pour la soustraction.

>Nous n'avons qu'une option : l'agrandir. Non par choix — par nécessité logique. Nous inventons :

1. **Création de zéro**, 0

```latex
la solution de n - n = 0. L'élément neutre, le retour au rien.
```

2. **Apparition des nombres négatifs**

```latex
pour que 3 - 5 = -2 ait un endroit où vivre.
```

**La Conséquence**

- La droite numérique subit une rupture de symétrie fondatrice. Elle n'a plus de commencement. Elle s'étend infiniment dans les deux directions, avec zéro comme centre autour duquel chaque positif a son parfait miroir négatif. Les entiers $\mathbb{Z}$ sont nés — clos pour la soustraction, enfin.

## Les Nombres Rationnels

**La Question**

- Peut-on annuler une multiplication ?

Le même raisonnement se répète.

- Si nous pouvons multiplier, pouvons-nous _annuler_ une multiplication ?

Nous inventons la division.

```latex
Exemples simples : 6 ÷ 2 = 3, 10 ÷ 5 = 2
```

Tout va bien tant que le résultat reste entier.

```latex
Mais 1 ÷ 2 n'a pas de réponse entière.
```

Aucun entier multiplié par 2 ne donne 1.

L'univers n'est pas clos, encore une fois.

Nous devons étendre l'univers.

```latex
Apparition des rationnels : a/b
```

**La Conséquence**

- La droite numérique devient dense.
- Entre deux entiers, une infinité de rationnels.
- ℚ est né — clos pour la division.

> Pour le ressentir :
> entre 1/2 et 1/3, prenez la moyenne 5/12.
> Entre 1/2 et 5/12, prenez 11/24.
> Le jeu continue.

## Apprivoiser l'Explosion — Les Logarithmes

**La Question**

- Étant donné le résultat, quel exposant l'a produit ?

La multiplication répétée devient exponentiation.

```latex
Définition : a^n = a × a × ... × a  (n fois)
```

La croissance peut devenir effarante.

```latex
Table illustrative :
n    10^n
1    10
3    1 000
6    1 000 000
9    1 000 000 000
12   1 000 000 000 000
```

Nous demandons l'inverse.

```latex
Résoudre : trouver x tel que 10^x = 1000
```

**La Conséquence**

- Naissance du logarithme.
- La multiplication devient addition via le log.

```latex
Formule clé : log(A × B) = log(A) + log(B)
```

Les échelles logarithmiques rendent lisible l'incommensurable.

```latex
Exemple : log_{10}(1000) = 3
```

## Le Mur de l'Impossible

**La Question**

- Quel nombre, élevé au carré, donne -1 ?

Les réels ℝ remplissent la droite.

```latex
Constantes célèbres : √2, π, e
```

Mais la droite impose une loi : tout carré réel ≥ 0.

```latex
Équation impossible : x^2 + 1 = 0
```

Aucune solution réelle.

Les mathématiciens ont choisi d'agir.

```latex
Postulat : existence d'un nombre i tel que i^2 = -1
```

**La Conséquence**

- Introduction de l'imaginaire.
- Nouvel alphabet pour résoudre l'irrésolu.

## Le Plan Complexe — Une Révolution en Deux Dimensions

**La Question**

- Où va i ?

i ne se place pas sur la droite.

Nous construisons un second axe.

```latex
Construction : plan (axe réel, axe imaginaire)
```

Les nombres complexes s'écrivent ainsi :

```latex
Z = a + i b    avec a, b ∈ ℝ
```

**La Conséquence**

- La multiplication se lit comme: rotation + homothétie.
- Multiplier par i = rotation de 90°.

## La Formule d'Euler — La Convergence Improbable

Cinq symboles se rencontrent.

```latex
0, 1, π, e, i
```

Ils viennent d'horizons différents et convergent.

```latex
Formule : e^{iπ} + 1 = 0
```

**La Conséquence**

- Une élégante synthèse de l'algèbre, de la géométrie et de l'analyse.
- Une invitation à poursuivre la promenade.

>Chaque étape montre une logique interne : poser une opération, constater une fermeture manquante, étendre l'univers par postulat ou construction, puis comprendre la nouvelle géométrie algébrique qui en découle.
