# Design Tokens

The visual language of the playground. Based on [Gruvbox](https://github.com/morhetz/gruvbox), defined as CSS custom properties in `packages/ui/src/styles/gruvbox-theme.css`.

All color tokens use the `oklch()` color space for better perceptual uniformity.

## Prose System

The `prose-playground` utility maps our design tokens to Tailwind's Typography plugin for consistent long-form content.

| Token               | CSS Variable Mapping |
| :------------------ | :------------------- |
| **Body Text**       | `var(--foreground)`  |
| **Headings**        | `var(--foreground)`  |
| **Links**           | `var(--primary)`     |
| **Bold**            | `var(--foreground)`  |
| **Bullets**         | `var(--border)`      |
| **Horizontal Rule** | `var(--border)`      |
| **Code Background** | `var(--card)`        |
| **Table Borders**   | `var(--border)`      |

## Usage in Code

Design tokens should be used via Tailwind utility classes or directly as CSS variables.

```tsx
// Using Tailwind (Preferred)
<div className="bg-primary text-primary-foreground p-4 rounded-lg">
  Primary Action
</div>

// Using CSS Variables
<div style={{ backgroundColor: 'var(--accent)', color: 'var(--accent-foreground)' }}>
  Custom Highlight
</div>

// Typography
<p className=" text-sm text-muted-foreground">
  v0.42.0-beta
</p>
```
