This is an invitation to a walk.

At every crossroads we will ask a simple question:

> What if ?

It is this question, asked again and again, that builds the whole architecture of numbers.

From the ground up, step by step.

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 01
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">The Natural Numbers</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">N</span>
  </div>
</h2>

<br />

We must begin somewhere.

> What if something exists ?

Call it "1". Then we add a single rule:

```text
If a number exists, its successor also exists.
```

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">1 + 1 = something</span>
</div>

The mechanism is as simple as it is tireless.

From one brick and one generative rule, an infinite set unfolds.

1 + 2 + 3 + ... infinity = N

<br />

Then, multiplication is not a new invention.

```text
It is just a compact notation for repeated addition.
```

1 + 1 = 1 x 2

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 02
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">Zero and the Negatives</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">Z</span>
  </div>
</h2>

<br />

> What if we undo the addition ?

```text
If we can add, we must be able to reverse the operation.
```

We invent subtraction. And for a while, everything holds.

<br />

Then someone dares to write:

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">1 - 1 = something</span>
</div>

Silence.

<br />

> The result should exist.

```text
The answer lives nowhere in our universe.
```

We have only one option: enlarge it.

We invent a new symbol for the void: "0"

And a new direction for the negatives: "-1".

We give the **negative numbers** somewhere to live. The integers.

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 03
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">The Rational Numbers</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">Q</span>
  </div>
</h2>

<br />

> What if we undo a multiplication ?

The same reasoning repeats. If we can multiply, we must be able to invert the operation.

We invent division. And once again at the beginning, everything works perfectly.

<br />

> But what if it doesn't divide cleanly ?

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">1 / 2 = something</span>
</div>

```text
The space between the integers is empty. There are no positions between one and two.
```

We invent new symbols to describe the pieces.

<br />

a/b, where a, b E Z and b != 0.

<br />

These new numbers need a home.

> The number line becomes dense in Q.

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 04
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">The Real Numbers</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">R</span>
  </div>
</h2>

<br />

From multiplication, we derive exponentiation.

3 x 3 x 3 = 3^3

```text
It is just a fancy notation.
```

<br />

> What if we undo the exponentiation ?

We build a tool for it: the square root.

sqrt(a) = b means that b^2 = a

For a while it was fine. Then a crack appeared in the geometry.

<br />

> What if we undo the square of 2 ?

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">sqrt(2) = something</span>
</div>

```text
We checked every fraction. No two integers can be divided to produce this ratio. It cannot be written.
```

The Real Numbers R are born to fill the infinite gaps.

sqrt(2), pi, e, 3.33333333... are real numbers

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 05
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">The Imaginary Unit</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">i</span>
  </div>
</h2>

<br />

We have noticed that when you square a real number, the result is always positive. Even if you
square a negative number, the result is still positive.

It enforces one absolute law of our reality:

Every real square is >= 0.

<br />

> What if we allow a square to be a negative number ?

```text
The equation x^2 + 1 = 0 has no real solution.
```

Mathematicians chose to act rather than retreat. They postulated the existence of a number i such that:

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">i^2 = -1</span>
</div>

Not a trick. Not a shorthand for something else. A legitimate new entity born from a necessity.

<br />

<br />

<h2 class="text-primary my-8 text-center text-2xl font-bold sm:text-3xl">
  <span class="text-muted-foreground/50 border-border inline-flex aspect-square w-fit items-center rounded-full border px-4 text-sm font-medium">
    Step 06
  </span>
  <div class="mt-16 flex items-center gap-4">
    <span class="shrink-0">The Complex Plane</span>
    <div class="bg-border h-px flex-grow"></div>
    <span class="text-accent shrink-0">C</span>
  </div>
</h2>

<br />

> Where is i ?

$i$ is not in the real world. It does not lie anywhere on our line. We need to be creative.

We create a new dimension.

```text
We visualize it as an axis perpendicular to the real axis.
```

<br />

And now with those two axes, the real number line extends into a plane. This is where the complex
numbers live: C

Every complex number takes the form:

<div class="not-prose border-secondary bg-secondary/5 text-secondary my-8 border-l-4 p-6 text-xl">
  <span class="block pl-4">Z = a + ib where a, b E R</span>
</div>

where "a" is the real part and "b" is the imaginary part. The line has become a plane.

<br />

> What if we bring them all together ?

Five symbols. Five different origins. One equation.

They come from different centuries, different problems, and different branches of mathematics.

They have no obvious reason to meet.

```text
And yet.
```
