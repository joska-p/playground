pg_lab is a design-first toolkit built on modern css. every interactive state is handled through css selectors, not javascript. gruvbox dark with blue primary keeps things warm yet focused.

  • css-only states via :has() and :focus-within
  • entry animations with @starting-style
  • dynamic theming with color-mix()
  • zero-js accordion, tabs, toggles
  • floating nav appears on scroll
  • orientation-based layouts
