# @repo/tailwind-config

Shared Tailwind CSS configuration featuring the Gruvbox theme.

---

## Installation

```bash
pnpm add @repo/tailwind-config
```

---

## Usage

Import the theme in your Tailwind config:

```js
// tailwind.config.js
import theme from "@repo/tailwind-config";

export default {
  theme,
};
```
