Hello, p5.js

Lesson 1 · 10 min

p5.js is a JavaScript library that makes drawing with code as natural as sketching with a pencil. You write a sketch — a small program that runs in a loop, creating motion and interaction.

The skeleton: setup + draw

Every p5.js sketch has two functions:

function setup() {
  createCanvas(400, 400);
}

function draw() {
  // runs 60 times per second
  circle(200, 200, 100);
}
  • setup() — runs once. Set up the canvas here.
  • draw() — runs 60 times per second (60 fps). Each call draws one frame. Clear it to animate, or let it accumulate.

The coordinate system

Unlike math class, (0, 0) is the top-left corner. X goes right, Y goes down.

Your first live sketch

Move your mouse over the canvas below. This sketch draws circles that follow your cursor and fade over time.

Shapes you'll use most

circle(x, y, diameter)
rect(x, y, width, height)   // (x,y) is top-left
ellipse(x, y, w, h)
line(x1, y1, x2, y2)
triangle(x1, y1, x2, y2, x3, y3)

Fill, stroke, noStroke

fill(r, g, b)          // RGB fill (0-255)
fill(r, g, b, alpha)   // with transparency
stroke(r, g, b)        // outline color
strokeWeight(n)        // outline thickness
noStroke()             // no outline
noFill()               // no fill

Try it yourself

Open your browser's console (F12) or edit the sketch above. Try changing p.background(0, 0, 95) to p.background(0, 0, 95, 30) — see how the trails smear instead of clearing completely. That's the seed of every particle system.

Key insight: p5.js is just JavaScript. The draw loop is the heart of everything — animation, particles, physics. Master the loop, and you can build anything.
