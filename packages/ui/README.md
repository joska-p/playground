# @repo/ui

Shared UI components built with React 19.

---

## Installation

```bash
pnpm add @repo/ui
```

---

## Usage

```tsx
import { Button } from "@repo/ui";

export function MyPage() {
  return <Button>Click me</Button>;
}
```
